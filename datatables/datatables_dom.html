Data sources

    DataTables 支持三种数据源显示：
  • DOM
  • JavaScript
  • Ajax
  • DOM
DataTables 初始化后，它会自动检查表格中的数据，如果存在即作为表显示的数据 （注意，如果你这时使用data或者ajax传递数据将不会显示），这是使用 DataTables 最简单的方法，渲染已经存在的table 注意,当使用DOM显示表,DataTables 将会把数据当做数组作为数据源(见上)。 DataTables还可以直接把 DOM 数据转化为自己的内部数据对象，示例如下：

Column 1 Column 2
Row 1 Data 1 Row 1 Data 2
Row 2 Data 1 Row 2 Data 2
